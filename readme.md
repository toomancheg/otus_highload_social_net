## Требования
Требуется разработать создание и просмотр анкет в социальной сети.


## Функциональные требования:

Простейшая авторизация пользователя.
Возможность создания пользователя, где указывается следующая информация:
``` 
Имя
Фамилия
Дата рождения
Пол
Интересы
Город
Страницы с анкетой. 
```

## Быстрый запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/toomancheg/otus_highload_social_net.git
cd social-network
```
2. Сбилдить проект
```bash
docker compose up
```
http://localhost:8080


3. Установить зависимости 
```bash
composer install
```
4. воспользоваться  Postman  коллекцией
